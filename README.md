# Prompt_Langchain

A Streamlit app that lets users interact with LangChain prompting using a Hugging Face LLM endpoint.

This project demonstrates how to build a simple research-prompt interface using:
- **LangChain** for LLM orchestration
- **HuggingFace Endpoints** for calling a model
- **Streamlit** for user interface
